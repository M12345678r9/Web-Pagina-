Objetivos de una página web

1Objetivo General

Crear páginas web con un diseño personalizado por medio de los conocimientos adquiridos en clases, incluyendo elementos para que la página web se convierta en un recurso informativo.

2Objetivo Específico I

Utilizar diferentes recursos para embellecer la página web e incluir la informacón correcta sobre el tema que seestá tratando.

3Objetivo Específico II

Identificar las funciones que poseen los diferentes elementos y codificarlos de manera correcta para lograr que su labor se realice de la forma correcta.
